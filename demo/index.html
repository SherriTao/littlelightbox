DESCRIPTION

Little LightBox is a jQuery Plign to display images in a "lightbox" over web page. It's lean and mean and such convenience for use, I dare say it will useful to enrich your web page. And also, maybe you could get something new from src code.

一款基本的、简洁的，十分易于使用的图片画廊插件。希望你可以从Little LightBox中获得和我同等多的乐趣和知识。

FEATURES

  • Customizable with various options

  • Responsive when size of window changes

  • Support keyboard control for closing lightbox and switch images

  • Support various effects to choice

EXAMPLES

Simple image gallery

Using default options to create lightbox gallery

Loop image gallery

Set option {loop: true}, you can get a loop gallery

Different effects for appear, disappear and change

You can choose various effects by setting "openEffect", "closeEffect" or "changeEffect" options

Different title type

You can choose different title type by setting title options helpers: {title: {type: 'inside'}}. 'inside' type title is out of image display area and 'over' type title is flow over display area.
